# Phase 4 — Individual Profile v1

## Read
- `docs/domain/DATA_AND_METRICS.md`
- relevant episode output docs

## Outcome
Produce explainable, role-filtered long-term individual profiles.

## Initial dimensions
- early growth stability;
- resource conversion efficiency;
- risk exposure management;
- objective readiness;
- optional fight contribution if reliable.

## Required
- per-match features first;
- comparable cohort;
- role separation;
- sample size;
- recency weighting;
- small-sample shrinkage;
- confidence/uncertainty;
- submetrics and evidence links.

## Acceptance
Small samples are not falsely precise, roles are not silently mixed, and values can be regenerated.

Stop after this phase.
