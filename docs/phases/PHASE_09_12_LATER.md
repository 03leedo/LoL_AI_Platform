# Phases 9–12 — Later Roadmap

These phases are intentionally deferred.

## Phase 9 — Live Client collector
Opt-in local collection, configurable polling, local queue, retries, idempotent sync, privacy controls, and final-match reconciliation.

## Phase 10 — Replay review
Seek to evidence timestamps, synchronize match time, show observation overlays, support notes, and handle unavailable replays.

## Phase 11 — Highlights
Generate configurable candidate windows. Users explicitly select public highlights. Prefer local clips or external links before paid storage.

## Phase 12 — Vision
Analyze only selected moments. Start with fixed HUD/minimap tasks and match-level dataset splits. Keep an API-only fallback. Never infer intent from vision.

Do not start these phases before earlier foundations pass.
